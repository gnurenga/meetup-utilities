## Video process unit: 
+ Add preface to videos
+ Trim the videos.
+ Join the videos
+ Trim the audio
+ Overlay audio on video.

### Requirements
+ Python3
+ ffmpeg

### Running the script

+ `pip3 install -r requirements.txt` (If requirements not met)

+ `python3 video-processing-utility.py`
